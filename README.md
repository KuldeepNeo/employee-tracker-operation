# Employee CRUD Operation

A full-stack Employee Tracker Operation that provides Create, Read, Update, and Delete (CRUD) functionality for managing employee records efficiently.

## 📋 Overview

This application allows users to:

* Add new employees
* View employee details
* Update employee information
* Delete employee records
* Manage employee data through an intuitive interface

The project demonstrates fundamental CRUD operations and follows a structured architecture suitable for learning and real-world applications.

---

## 🚀 Features

* ✅ Create Employee Records
* ✅ Read/View Employee Information

## 🔗 API Endpoints

| Method | Endpoint          | Description         |
| ------ | ----------------- | ------------------- |
| GET    | `/employees`      | Get all employees   |
| GET    | `/employees/{id}` | Get employee by ID  |
| POST   | `/employees`      | Create new employee |
| PUT    | `/employees/{id}` | Update employee     |
| DELETE | `/employees/{id}` | Delete employee     |
